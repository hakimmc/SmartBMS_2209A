enum \mbox{\hyperlink{d2cc__lib_8h_abc354ec37af201f0ebb8f4bde71ddd57}{Battery\+Chemistry\+\_\+enum}}}



Enum for representing signal values for the Battery\+Chemistry signal. 

This enum maps each value in the signal\textquotesingle{}s value table to a corresponding identifier. It is used to interpret the raw data received from the CAN bus and provides human-\/readable names for each value.
